一、首先编辑coser1.0文件 搜索api 找到deepseek api（聊天）填写进去你的api 和agnes api（识图）然后确保你的基本的ai聊天功能能使用。
二、然后本地部署npm （不会问ai，也可以不做 也能玩）。1. 打开浏览器，访问：https://nodejs.org/
2. 点击绿色的 LTS（长期支持版） 大按钮，会自动下载安装包
3. 下载完成后，双击安装包，一路点 “下一步”（Next），直到安装完成
打开终端（命令行）
两种方法任选一种：
    方法 A：在项目文件夹空白处，按住 Shift 键 + 鼠标右键 → 选择 “在此处打开 PowerShell 窗口”
    方法 B：按 Win + R，输入 cmd 回车，然后用 cd 命令切换到项目目录
    在刚才打开的终端里，输入下面这行命令，然后按回车：npm install
    启动项目:在同一个终端里，输入：npm start
    或者（如果上面不行）：node server.js
    打开 Chrome/Edge/任何浏览器，在地址栏输入：http://localhost:3000,就能本地记录聊天记录了
***最重要***三：打开data.js文件 里面可以更改人物的名字 头像 性格 以及个人主页的背景图 以及个人作品的显示。这个直接决定了你的游玩体验。
比如作品的播放路径改成：asset/video/你的视频的名字.mp4 就会播放你asset文件夹里 video文件夹里的一个叫“你的视频名字”的视频
